# Bomber

A simple Unity project.

## Unity Version
This project was built with **Unity 2022.3.14f1**. Open the project using this version or a compatible release via Unity Hub.

## Opening the Project
1. Launch **Unity Hub**.
2. Click **Add** and select this project folder.
3. Ensure Unity Hub uses version **2022.3.14f1** to open it.

## Build and Run
1. After Unity loads the project, go to **File > Build Settings**.
2. Select your target platform and click **Build** or **Build and Run**.
3. The compiled build will appear in the folder you choose. Run the generated executable to play the game.
